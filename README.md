# Sensor Based Plant Protection Using Arduino

## Overview
This project is developed using Arduino to protect plants by monitoring environmental conditions. The system uses different sensors to detect rain, light, and other environmental changes, then automatically controls the protective cover using servo motors.

## Features
- Rain Detection
- LDR (Light) Detection
- Automatic Protective Cover Control
- Arduino-based Automation

## Components Used
- Arduino Uno
- Rain Sensor
- Rain Drop Sensor
- LDR Sensor
- Servo Motor
- Jumper Wires
- Power Supply

## Software Used
- Arduino IDE
- Embedded C
